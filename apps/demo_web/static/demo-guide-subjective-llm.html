6）主観リスク（LLM エージェント）

R_subj はモデル上、客観より楽観に振れます。体験では右カラムの引率人格・パーティチャットが「人的側の読み」のメタファです。数値の正は常に Pythonです。

項目 内容
引率人格選択 リセット時のシナリオ初期パラメータと、Ollama ON 時の人格テキストの両方に効きます。操作は 2）操作と引率。
Ollama 引率 ON 時は各ステップで trek／rest を選び、朔の発言として理由を表示。隊員は状態に応じた短文。OFF 時も三人の掛け合いは出ますが朔は定型。
中止コーチ LLM 介入→中止で avoided になったあと、コーチ文言が出る場合があります（Ollama 依存）。プロンプトで提示数値の捏造を禁止しています。
