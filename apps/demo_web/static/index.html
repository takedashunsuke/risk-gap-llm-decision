このデモについて

          
        

        

          

            R_obj（客観リスク）とR_subj（主観リスク）のずれ・Gap を、左の主要指標・内訳と中央のグラフ・パーティ登山で確認できます。
          

          

            ステップ進行で場所の背景が変わり、上限到達後は「続行」か「LLM 介入 → 中止」を選べます。右カラムでは引率エージェント設定とチャット、中止時のコーチメッセージを表示します。
          

          

            地名とエンディングの対応は docs/design/rp_simulation.md を参照してください。
